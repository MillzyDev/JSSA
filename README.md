# JSSA
Java Score Saber API Wrapper

## Requirements
> JDK 16 (Recommended)
>
> Google Gson v2.8.7

## Installation
### Maven
```xml
<dependencies>
  <dependency>
    <groupId>dev.MillzyG</groupId>
    <artifactId>JSSA</artifactId>
    <version>1.0</version>
  </dependency>
  <dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.8.7</version>
  </dependency>
</dependencies>
  ```
